# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it responsibly. **Do not disclose security vulnerabilities publicly until they have been addressed.**

To report a security issue, please email: [contact@datafoundation.lk](mailto:contact@datafoundation.lk)

## What to Include

When reporting a vulnerability, please include:

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Any suggested fixes (if available)

## Response Timeline

- We will acknowledge your report within 48 hours
- We will provide an initial assessment within 7 days
- We will work with you to understand and address the issue promptly

Thank you for helping keep Lanka Data Foundation projects secure.
